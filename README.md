## ⚡ Overview

**Meetuoo** is a premium, high-performance platform designed to revolutionize how teams handle meeting data. By leveraging state-of-the-art AI, Meetuoo saves technical teams hours of manual analysis, providing clean, synthesised insights through a stunning, "blueprint-inspired" technical interface.

> Save hours of meeting analysis with clean, ready-to-use AI insights that just work — modern, responsive, and built for speed.

<br />

## ✨ Key Features

- 🧠 **AI-Driven Intelligence** – Deep analysis of meeting transcripts using OpenAI and Groq-powered models.
- 🔭 **Synthesis & Discovery** – Automatically extract action items, patterns, and mission-critical data points.
- 🔐 **Enterprise-Grade Auth** – Secure, seamless authentication flow powered by Supabase SSR.
- 📐 **Technical UI/UX** – A "blueprint" aesthetic with grid-based layouts, dark mode saturation (`#121212`), and fluid micro-animations.
- ⚡ **Optimised Performance** – Built on the cutting edge with Next.js 16, React 19, and Tailwind CSS 4.
- 📧 **Communication Layer** – Integrated email workflows using Resend.
- ☁️ **Media Management** – Cloudinary integration for scalable asset handling.

<br />

## 🛠️ Tech Stack

- **Framework:** [Next.js 16 (App Router)](https://nextjs.org/)
- **Runtime:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Backend/Auth:** [Supabase](https://supabase.com/)
- **AI Models:** OpenAI SDK & Groq SDK
- **Icons:** Lucide React
- **Validation:** Zod
- **Infrastructure:** Cloudinary (Media), Resend (Email)

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- [Bun](https://bun.sh/) (recommended) or npm/yarn
- A Supabase account and project

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/meetuoo.git
   cd meetuoo
   ```

2. **Install dependencies:**
   ```bash
   bun install
   # or
   npm install
   ```

3. **Environment Setup:**
   Create a `.env.local` file in the root directory and add your credentials:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   OPENAI_API_KEY=your_openai_key
   GROQ_API_KEY=your_groq_key
   CLOUDINARY_URL=your_cloudinary_url
   RESEND_API_KEY=your_resend_key
   ```

4. **Run the development server:**
   ```bash
   bun dev
   # or
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## 📐 Design Philosophy

Meetuoo follows a **Technical Minimalist** aesthetic:
- **Background:** `#121212` with subtle noise texture.
- **Accents:** High-fidelity blue/indigo gradients.
- **Grids:** Blueprint-inspired vertical and horizontal lines to represent precision.
- **Typography:** Display-heavy, uppercase headers with a focus on Geist and Poppins.
